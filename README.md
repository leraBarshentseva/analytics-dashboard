# Интерактивная аналитическая панель | Тестовое задание

Это реализация тестового задания на позицию Frontend-разработчика.

**Демо-ссылка:** [Посмотреть демо](https://lerabarshentseva.github.io/analytics-dashboard/)

---

## Основные возможности

- **Динамическая генерация данных:** Вся информация в таблице генерируется на лету с помощью JavaScript.
- **Интерактивный график:** График мгновенно обновляется при выборе строки в таблице.
- **Условное форматирование:** Ячейки в таблице меняют цвет в зависимости от значений (например, рост или падение показателей).
- **Адаптивная верстка:** Таблица корректно отображается на мобильных устройствах.

## Стек технологий и инструменты

- **HTML5** (Семантическая верстка)
- **CSS3** (CSS Variables, Media Queries для адаптивности)
- **JavaScript (ES6+)** (Vanilla JS, без фреймворков)
- **ApexCharts.js** (Библиотека для отрисовки интерактивных графиков)
- **GitHub Pages** (Для деплоя)

## Как запустить локально

1.  Клонируйте репозиторий:  
    `git clone https://github.com/leraBarshentseva/analytics-dashboard`
2.  Перейдите в папку проекта:  
    `cd analytics-dashboard`
3.  Откройте файл `index.html` в вашем браузере.
